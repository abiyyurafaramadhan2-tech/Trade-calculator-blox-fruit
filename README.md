# Blox Fruits Trade Calculator

A web-based trade calculator for Blox Fruits, helping players evaluate fair trades by comparing fruit values, prices, and demand.

## Features
- Select from a list of popular fruits with accurate values
- Real-time comparison of value, price, and demand
- Trade fairness indicator (Overpay, Fair, Gain)
- Swap sides and reset functionality
- Fully responsive design for mobile and desktop

## Data Source
Values based on community-driven data from Blox Fruits trading communities.

## Deployment
Deployed on Vercel. To run locally:
1. Clone the repository
2. Open `index.html` in any modern browser

## Technologies
- HTML5
- CSS3 (Flexbox, Grid, responsive)
- Vanilla JavaScript (ES6)
